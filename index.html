Basic Operations

  • Select Folder: Click "Select Folder" to load videos.
  • Switch Video: Use left/right arrow keys or click on video name.
  • Playback: Space to play/pause, fast forward/rewind buttons.

Video Trimming & Export

  • Set trim points: Drag white handles on timeline or use "Set Start/End".
  • Export: Click "Export" to choose quality & frame rate. Tasks run in background.

Shortcuts

  • ← / → : Switch video
  • Space : Play/Pause
  • Ctrl + +/-/0 : Zoom
  • Ctrl+Z : Undo trim

FFmpeg Installation (Required)

This software requires FFmpeg for video processing. Please install before using.

Installation Steps

  1. Gyan FFmpeg Builds
  2. ffmpeg-release-full.7z
  3. ffmpeg.exe
Your Program Folder/
├─ VideoManager.exe
└─ ffmpeg/
    └─ ffmpeg.exe

Note: No need to modify system PATH. After installation, restart this app. Status bar will show "FFmpeg ready".

Advanced (Optional)

If you want to use FFmpeg globally, add its bin folder (e.g., C:\ffmpeg\bin) to system PATH environment variable.
